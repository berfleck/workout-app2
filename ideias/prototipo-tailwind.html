Dicas

Ao gerar, exercícios marcados como REF serão evitados. Novos exercícios ganharão badge novo.

O que mudou vs. seu app atual

  • • 3 colunas em vez de 1 (960px): aluno+histórico à esquerda, config no meio, refs à direita
  • • Configurações gerais colapsada (abre só quando precisa ajustar)
  • • Hierarquia em acordeões, não 3 colunas de checkboxes sempre abertas
  • • Tabs visuais de treino (estilo nativo) em vez de botões com background hardcoded
  • • Refs sempre visíveis à direita, não enfiadas dentro do fluxo da config
  • • Botão "Gerar" sticky na sidebar — não precisa rolar

Custo real

  • • Tailwind via CDN: 1 linha no base.html, zero build step
  • • Remove ~300 linhas de CSS do base.html
  • • Não mexe no backend: Flask, rotas, HTMX, lógica de geração — tudo intacto
  • • Prazo honesto para chegar neste layout: 3-5 dias de refactor de templates
  • • Este protótipo é estático: os botões não fazem nada — é só visual
